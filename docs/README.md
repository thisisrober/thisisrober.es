## Quick Start

```bash
# Install dependencies
npm install

# Initialize database (first time only)
npm run db:init

# Start development servers
npm run dev
```

- Frontend: http://localhost:5173
- API: http://localhost:3001
- Dashboard: http://localhost:5173/dashboard
